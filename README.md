# Krithin Kamineni — Personal Portfolio

Personal portfolio website built with Next.js 15, TypeScript, and Tailwind CSS.

## Stack

- **Framework** — Next.js 15 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS
- **Fonts** — DM Serif Display, DM Sans, JetBrains Mono (Google Fonts)

## Features

- Animated hero with typewriter effect and floating headshot
- Draggable photo strip
- Experience, Projects, Organizations, and Skills sections
- Separate `/coursework` page with semester-by-semester course breakdown
- Floating pill navbar with sliding active indicator
- Cursor-following amber glow
- Scroll-based fade-in animations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.
